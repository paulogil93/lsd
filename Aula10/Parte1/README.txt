README
--------------------------------------------------------------------------------------------------
Este exercício foi resolvido usando dois divisores de frequencia, um a 1Hz e outro a 10Hz,
que serão ativados através de um switch. Se o switch for igual a 0, iremos incrementar o contador
a 1Hz, se for igual a 1, o contador incrementa a 10Hz. Isto foi possível usando um Mux 2:1.
